# Notiq

My friend mentioned that Notion feels a bit cluttered nowadays, so I built this. It's a minimal, offline-first workspace just for us.

**It's hosted on this website:** [https://notiq.iamleander.me/](https://notiq.iamleander.me/)

## Features

- Clean, block-based text editor (type `/` for commands)
- Global Command Palette (`Cmd + K` or `Ctrl + K` to search pages, toggle themes, etc.)
- Offline storage (everything saves locally to your browser)
- Simple JSON export/import for backups
